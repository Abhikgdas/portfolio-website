#  Portfolio Website

This is my personal portfolio website built using HTML, CSS, and basic JavaScript. It showcases my projects, skills, and contact information.

---

##  Project Structure

- `index.html` → Main webpage  
- `style.css` → Styling of the website  
- `images/` → Contains all images used  
- `cv/` → Contains my resume  
- `christmas/` → HTML project (Greeting Card)

---

##  Features

- Responsive design (works on mobile & desktop)
- Navigation bar with smooth scrolling
- About, Services, Projects, and Contact sections
- Social media links
- Download CV option
